<h1>Egg Timer App 🥚</h1>

Prosta aplikacja do mierzenia czasu gotowania jajek z graficznym interfejsem użytkownika. Umożliwia wybór stopnia ugotowania jajka.

<h3>✨ Funkcje</h3>

<li>Wybór stopnia ugotowania:</li>

Jajko na miękko (5 minut)

Jajko na półmiękko (7 minut)

Jajko na twardo (10 minut)

<li>Odliczanie w czasie rzeczywistym</li>

<h3>🛠️ Wymagania</h3>

<li>Java 17 lub nowsza</li>

<li>JavaFX SDK</li>

<li>IntelliJ IDEA (lub inny kompatybilny IDE)</li>

<h3>📁 Instalacja</h3>

1. Pobierz JavaFX SDK

<li>Link do pobrania JavaFX</li>

<li>Wybierz wersję dla swojego systemu (dla Mac M1/M2: aarch64)</li>

2. Dodaj bibliotekę JavaFX w IntelliJ

<li>File → Project Structure → Libraries → + → Dodaj folder lib z wypakowanego SDK</li>

3. Skonfiguruj opcje VM

<li>Run → Edit Configurations...</li>

<li>W polu VM options dodaj:</li>

<i>--module-path ~/javafx-sdk/lib --add-modules javafx.controls,javafx.fxml,javafx.media</i>

<h3>⚙️ Jak uruchomić</h3>

1. Skompiluj projekt w IntelliJ.

2. Uruchom klasę EggTimerApp z ustawionymi VM options.

3. Wybierz stopień ugotowania i rozpocznij odliczanie!
